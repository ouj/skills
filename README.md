# Skills
My commonplace book for skills. No skill issues.

## Installation

```sh
npx skills@latest add ouj/skills/<skill-name>
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [git-pr](git-pr/SKILL.md) | Create or update a GitHub pull request for the current branch safely, including stacked-branch handling and duplicate-PR avoidance. |
| [git-commit](git-commit/SKILL.md) | Create intentional local git commits safely without pushing, with conservative staging and default-branch protection. |
| [git-sync](git-sync/SKILL.md) | Sync a feature branch onto the latest remote default branch with rebase, conflict handling, and safe force-push. |
| [pr-stack](pr-stack/SKILL.md) | Manage stacked pull requests on plain git and GitHub CLI, including parent tracking, restacking, submit/update, and sync workflows. |
| [rubber-duck](rubber-duck/SKILL.md) | Explore coding and design problems through one-question-at-a-time dialogue; compare clean approaches, align on interfaces and system interactions, and write a design doc before implementation. |
